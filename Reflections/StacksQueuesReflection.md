### MyStack\<T\>:

MyStack uses the two stacks for undoing and redoing the text that was written or added to the string.
The method time complexity for pushing the new text into the stack for undo and redo history was O(1)
amortized, while popping and peeking into the recent history. The common use cases would be the word
processing program like Libreoffice, Microsoft office, or other word processing, any text editor and code editor.


### MyQueue\<T\>:

MyQueue uses a single queue to add a job with a name, processing and peeking into the next job.
The method time method for enqueuing into the job with a name was O(1) amortized. for the processing
and peeking into the next job, the time complexity was O(1). The common use cases for the queues was the
printing from the pdf file into a physical paper form, downloading stuff from online.
